## Core Philosophy: User Ownership

This project is built on a principle of user ownership. It is not a preference — it is a constraint that rules out entire categories of architecture.

- **The user owns the program.** No subscription, no licence server, no cloud activation. If you have the file, it runs.
- **The user owns their data.** Nothing is sent anywhere without explicit user action. No telemetry, no phone-home analytics, no cloud sync.
- **This applies especially to schools.** Educators and institutions should own the software they run and the content they create with it — not Microsoft, not Google, not any platform that can revoke access, change pricing, or harvest student data.

A copy on a USB drive that runs in a room with no internet is a success condition, not an edge case.
