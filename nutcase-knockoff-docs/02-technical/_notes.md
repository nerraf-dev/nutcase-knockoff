In terms of how it works, take a 2 player game:
Game Start -> P1 can reveal a slider or guess. (All are hidden so unlikely ot guess!)
P1 revelas a word - > Play plasses to P2

P2 chooses to reveal or guess -> P2 reveals -> Play passes to P1

P1 Guesses -> Guess incorrect --> P1 is frozen out. -> Pass to P2

P2 is 'last player standing' -> P2 is given a penalty free guess.

p2 is corrcect - they score, if incorrect no score. -> Next question, P2 Starts. 


For more than 2 players, after a player guesses incorrectly they are frozen out and play continues as normal. So in the case of a 3p game and P1 is frozen play moves to p2 who can revel or guess and so on.
